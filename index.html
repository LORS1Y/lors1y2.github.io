System Administrator

Директор

CISA System Administrator

Personnel performing this role may unofficially or alternatively be called:

  • Local Area Network (LAN) Administrator
  • Security Administrator
  • Server Administrator
  • System Operations Personnel
  • Website Administrator
  • Identity Access Manager
  • UNIX/Windows System administrator
  • Application Security Administrator
  • System Security Administrator

Skill Community: IT
Category: Operate and Maintain
Specialty Area: Systems Administrator
Work Role Code: 451

Core Tasks

  • Conduct functional and connectivity testing to ensure continuing operability. (T0029)
  • Develop and document systems administration standard operating procedures. (T0063)
  • Maintain baseline system security according to organizational policies. (T0136)
  • Manage accounts, network rights, and access to systems and equipment. (T0144)
  • Plan, execute and verify data redundancy and system recovery procedures. (T0186)
  • Install, update and troubleshoot systems/servers. (T0418)
  • Comply with organization systems administration standard operating procedures. (T0458)
  • Implement and enforce local network usage policies and procedures. (T0461)
  • Manage system/server resources including performance, capacity, availability, serviceability, and recoverability. (T0498)
  • Monitor and maintain system/server configuration. (T0501)
  • Perform repairs on faulty system/server hardware. (T0515)
  • Troubleshoot hardware/software interface and interoperability problems. (T0531)

Core Competencies

  • Identity Management
  • Information Technology Assessment
  • Operating Systems
  • System Administration
  • Systems Integration

Core Knowledge

  • Knowledge of organizational information technology (IT) user security policies (e.g., account creation, password rules, access control). (K0158)
  • Knowledge of measures or indicators of system performance and availability. (K0053)
  • Knowledge of performance tuning tools and techniques. (K0064)
  • Skill in monitoring and optimizing system/server performance. (S0155)
  • Skill in operating system administration (e.g., account maintenance, data backups, maintain system performance, install and configure new hardware/software). (S0158)
  • Knowledge of systems administration concepts. (K0088)
  • Knowledge of virtualization technologies and virtual machine development and maintenance. (K0130)
  • Knowledge of system administration, network, and operating system hardening techniques. (K0167)
  • Skill in correcting physical and technical problems that impact system/server performance. (S0144)
  • Skill in recovering failed systems/servers (e.g., recovery software, failover clusters, replication, etc.). (S0157)
  • Knowledge of principles and methods for integrating system components. (K0346)

How To Apply

When applying for this cyber roles, please review the information above and update your resume to align your experience with the listed competencies. Your resume must also show demonstrated IT-related experience in:
  • Attention to Detail
  • Customer Service
  • Oral Communication
  • Problem Solving
